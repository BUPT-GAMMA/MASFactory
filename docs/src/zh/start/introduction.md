# 项目简介

**MASFactory** 是一个面向 **VibeGraphing** 的多智能体工作流编排框架：围绕“从意图到结构”的工作方式，帮助你快速形成工作流草案，并持续迭代到可运行、可复用、可追踪的多智能体系统。

它把结构设计、流程编排、可视化预览与运行追踪整合为一条统一路径，适用于复杂、多阶段、多角色协作的 Agentic Systems。

## 推荐学习路径

- 第一步：先看 [第一行代码](/zh/start/the_first_code) 跑通最小示例。
- 第二步：阅读 [声明式 vs 命令式](/zh/start/declarative_vs_imperative) 学习手动搭建多智能体系统的两种开发范式。
- 第三步：进入 [渐进式教程](/zh/progressive/)，分别用「声明式 / 命令式 / VibeGraph」三条路径构建简化版 ChatDev。

## 为什么选择 MASFactory


- 🎨 **VibeGraphing：自然语言驱动的编排体验：** 从意图描述开始，快速生成流程结构草案，并在可视化环境中逐步完善与收敛，让多智能体系统的设计与迭代更直观、更高效。
- 🧩 **图结构“搭积木”：从简单到复杂的工作流扩展：** 以图为核心组织能力模块，把常见的流程模式沉淀为可组合单元，支持从小型流程平滑演进到复杂的多阶段、多分支、多循环系统。
- 🔍 **可视化与追踪：把结构与运行状态“看得见”：** 提供结构预览与运行追踪能力，帮助你更快理解系统行为、定位流程问题，并以更低成本迭代优化。
- 🧠 **上下文协议适配：让信息组织更有序：** 面向多来源上下文（记忆、检索、外部知识与工具结果）提供统一的组织方式，让上下文更可控、可管理、可追溯，支撑更稳定的系统表现与持续演进。
